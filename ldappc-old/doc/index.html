Ldappc Documentation Roadmap

This document describes the other documents in this package related to the Ldappc program. If you are viewing this document as part of the Javadoc documentation, after selecting one of the documents, use the back button to return to the menu that references all of the documents in order to navigate to the next document.

In addition to the documents included in this package, the RFQ document Request for Quote - LDAP Connector for Grouper & Signet is important to an understanding of the application. For those interested in a complete picture of the product, it is the first document that should be read.

If you desire information about LDAP in general, go to LDAP Guide.

If you desire information about the code architecture, go to Architecture.

If you desire more detailed design information, go to Design.

If you desire information on deploying the code for development purposes or as an end-user product, go to Deployment Guide.

If you desire information on using the code go to User Manual.

If you desire detailed information about the Java classes, go to Javadoc. You may need to run "ant javadoc" to generate the javadocs.
