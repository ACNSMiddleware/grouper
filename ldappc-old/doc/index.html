Ldappc Documentation

This document describes the other documents in this package related to the Ldappc program.

In addition to the documents included in this package, the RFQ document Request for Quote - LDAP Connector for Grouper & Signet is important to an understanding of the application. For those interested in a complete picture of the product, it is the first document that should be read.

See:

LDAP Guide for information about LDAP in general.
Architecture for information about the code architecture.
Design for more detailed design information.
Deployment Guide for information on deploying the code for development purposes or as an end-user product.
User Manual for information on using the code.
Javadoc for detailed information about the Java classes. You may need to run "ant docs" to generate the javadocs.
