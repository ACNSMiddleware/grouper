Ldappc Release Notes

Changes in version 1.1

Updated grouper API from 1.0.1 to 1.3 snapshot. Updated subject API from 0.2.1 to 1.3.1. Made numerous performance changes to increase speed. This involved resolving:

MCO-2LDAPpc should have LDAP connection pooling on by default
MCO-6LDAPpc does not accurately provision groups under heavy load
MCO-7LDAPpc error logs report spurious null DTO
MCO-8Update Ldappc to use better child group query filter
MCO-11LDAPPC fails when trying to provision very large groups

It is quite possible that MCO-6 and MCO-11 have not been completely fixed.
