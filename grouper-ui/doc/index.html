The Grouper User Interface (UI)

The Grouper UI is a Java web application which provides a non-programmatic means of exercising the Grouper API. The Grouper UI can be used to:

  • create groups and stems
  • delete groups
  • assign members to groups
  • grant and revoke privileges for groups and stems
  • browse or search for existing groups and stems
  • edit existing groups and stems
  • list group memberships
  • list Subjects* with a specific privilege for a group or stem
  • search subjects, and, for a selected subject:
    • list groups where the subject is a member
    • list groups or stems where the subject has a selected Access or Naming privilege

* Typically group members and privilegees will be people or other groups, however, any entity can be a member of a group or a privilegee if there is an appropriate Subject API implementation for it.

Running the Grouper UI

If you have downloaded the QuickStart distribution, follow the instructions in the QuickStart README.html file, If you have checked out the grouper-ui module from CVS, follow the instructions for Configuring the Grouper UI from CVS.

Customising the Grouper UI

The QuickStart distribution contains UI customisations. This document explains those customisations - which can be used as the basis for your own site-specific customisations.

Customisations may be limited to branding, page layout and text display, but can also include integrating authentication schemes and adding completely new functionality to integrate Grouper with other systems
