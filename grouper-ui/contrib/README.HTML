Grouper UI Contributed code

Code Description
quick-start Provides a means of loading data into the Grouper repository. It was developed to enable the reproducible production of a known repository for use with the Grouper QuickStart distribution, but may have a more general use for sites wishing to load data.
struts-patch Ensures that the current Strut's 'module' is maintained in links and internal forwards / includes. This enables Strut's modules to be used to provide multiple user interfaces in one web application instance
yale-cas-auth Provides the CAS client library and configures Servlet API Filters to manage authentication
