Running the HSQLDB command line utility
---------------------------------------

The following instructions are for an MS Windows command
prompt, but should adapt easily to a Linux/Unix/OSX shell.


1. Start the Signet QuickStart DB...

    cd  C:\Documents and Settings\ddonn\signet-qs-1.3.0rc03\demo
    start_hsqldb

2. To run the HSQLDB command line utility, open a new DOS command window, then...

    cd  C:\Documents and Settings\ddonn\signet-qs-1.3.0rc03\demo
    set_env  HSQLDB
    java  -jar  %HSQLDB_EXEC%  --rcfile  %HSQLDB_RC_FILE%  localhost-sa

3. When finished, shutdown the DB...

    shutdown_hsqldb.bat
