These files are the startup / shutdown scripts for the QuickStart Demo.
They are copied to the 'demo' directory by the Ant build_QS.xml script.

The apache-tomcat installer file (apache-tomcat-5.5.16.zip as of this writing)
is only used to create and run the QuickStart Demo.
