Grouper client institution distribution:

- Here is information on grouperClient:

https://spaces.internet2.edu/display/Grouper/Grouper+Client

- Generally:

1. configure the config file: grouper.client.properties with your credentials
2. make sure you have Java 1.5+ (same as Java 5+)
3. run and get the usage: java -jar grouperClient.jar
4. if you are running LDAP or WS, try one of those commands from usage
